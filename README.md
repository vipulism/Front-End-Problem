# Front-End-Problem
 

Problem : You are given a sample JSON data of payments consisting of PaymentId,OrderDate,MerchantId,CustomerEmail,Amounta and PaymentStatus.

PaymentStatus could be one these -> ('Initiated','Failed','Dropped','Success','Refunded').

You need to build a UI for showing this data. You can keep the UI simple or creative depends on you.

Results should be shown in paginated way.

  User should be able to perform following operations :-
  
      1. Sorting on PaymentId,OrderDate and Amount.
      2. Filtering on PaymentStatus.
      3. User can select how many results he wants to see per page. So the pagination will change accordingly.

Please find attached the JSON required to implement the above problem statement.

Note : Please use AngularUI Route to implement routes
